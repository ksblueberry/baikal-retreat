# Baikal / Olkhon — Retreat Landing

Одностраничный лендинг выездного оздоровительного ретрита на острове Ольхон (Байкал). Editorial-стиль, тёплая палитра, фоновые фото мест ретрита, плавные анимации, адаптивная вёрстка.

**Live:** _добавьте ссылку после деплоя_

---

## О проекте

Страница-презентация ретрита 10–20 августа 2026 года. Группа до 12 человек. Йога, дыхательные и телесные практики, экскурсии к местам силы, баня на дровах, вечера у костра.

Сайт служит двум задачам: рассказать о формате и ведущих, и довести посетителя до контакта (WhatsApp / Telegram / звонок).

## Стек

- HTML5, CSS3, vanilla JavaScript — без фреймворков и сборщиков
- Google Fonts: Fraunces, Cormorant Garamond, Inter
- IntersectionObserver для scroll-reveal и счётчиков
- CSS Grid + Flexbox
- CSS `backdrop-filter`, кастомные свойства, `clamp()` для типографики

## Структура

```
.
├── index.html        — разметка, семантические секции
├── styles.css        — стили, адаптив, анимации
├── script.js         — лоадер, скролл-ревил, счётчики, параллакс, меню
├── images/           — фотографии ретрита и ведущих
└── README.md
```

4. **Программа** — 8 карточек практик

## Адаптив

- `≥ 1100px` — десктоп с фиксированной боковой навигацией
- `≤ 1100px` — планшет, hamburger-меню сверху
- `≤ 780px` — мобильный: переработанная типографика, стек-раскладка, увеличенные тап-зоны
- `≤ 420px` — малые экраны, плотная упаковка
- Отдельные правила для `hover: none` (тач-устройства) и `prefers-reduced-motion`

## Запуск локально

Статика — достаточно открыть `index.html` в браузере. Для относительных путей к фото лучше поднять простой сервер:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Фотографии
